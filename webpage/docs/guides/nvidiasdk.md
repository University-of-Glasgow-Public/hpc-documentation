# Nvidia HPC SDK

The NVIDIA HPC Software Development Kit (SDK) includes the proven compilers, libraries and software tools essential to maximizing developer productivity and the performance and portability of HPC applications.

If Nvidia HPC SDK is available on your system (check via `module avail`) access for will most probably be restricted, you’ll need to check the software license terms, contact us via [Ivanti](https://glasgow.saasiteu.com/Modules/SelfService/#serviceCatalog/request/AFB25E75ED5E40E4BDF84FDEE6108945) to to arrange that.
